# Non-Goals

For v0.1 this project explicitly does not support:

- Autonomous real-world action.
- Self-modifying model weights.
- Unsupervised internet access.
- Financial, medical, legal, or physical-world execution.
- Hidden memory mutation.
- Unlogged packet flow.
- Production-grade agent autonomy.
- Claims that the toy verifier is a truth oracle.
- Long-term personal memory about real users.
- Live external integrations.
